# 🇯🇵 日本株 ネットキャッシュ比率スクリーナー

日本株の中から「ネットキャッシュ比率」が高い銘柄をスクリーニングするWebアプリです。

## ネットキャッシュ比率とは

```
ネットキャッシュ = 流動資産 + 投資有価証券 × 70% − 負債合計
ネットキャッシュ比率 = ネットキャッシュ ÷ 時価総額
```

比率が1.0を超える銘柄は、時価総額よりも保有現金・資産が多いことを意味します。

## 機能

- JPX上場銘柄リストの自動取得
- 市場区分（プライム／スタンダード／グロース）でフィルタリング
- 取得データのキャッシュ保存（中断・再開対応）
- 財務推移グラフの表示
- 銘柄ごとのメモ・評価機能
- CSVダウンロード

## 使い方

### 1. 依存ライブラリのインストール

```bash
pip install streamlit yfinance pandas requests plotly openpyxl
```

### 2. アプリの起動

```bash
streamlit run app.py
```

### 3. スクリーニング

1. サイドバーでネットキャッシュ比率の閾値と対象市場を選択
2. 「スクリーニング開始」ボタンをクリック
3. 結果をCSVでダウンロードまたは銘柄詳細を確認

## ファイル構成

| ファイル | 説明 |
|---|---|
| `app.py` | メインアプリ |
| `data_j.xls` | JPX銘柄リスト（自動ダウンロード） |
| `net_cash_cache.pkl` | 取得データのキャッシュ |
| `memo.json` | 銘柄メモ・評価データ |
